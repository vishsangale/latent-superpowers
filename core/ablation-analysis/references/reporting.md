# Reporting

## Good ablation outputs

- ranked run table
- grouped variant table with count, mean, min, max, stddev
- explicit baseline or top-variant comparison
- one local plot
- short findings section with caveats

## Findings template

Use this shape when summarizing:

1. State the metric and ranking direction.
2. Name the winning run or variant.
3. Name the strongest varying factors.
4. Note the sample count and any instability.
5. Say what to run next.
